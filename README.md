# JD Barber Shop ✂️

Landing page de **JD Barber Shop**, diseñada y personalizada para mostrar los servicios de barbería, horarios y contacto.

## Descripción

Este proyecto es una página web estática que funciona como carta de presentación digital de la barbería.  
Incluye:

- Sección principal con presentación de la marca.
- Información de servicios.
- Galería e imágenes de apoyo.
- Botones de contacto y agendamiento por WhatsApp.

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- Imágenes y recursos estáticos propios de la marca JD Barber Shop.
